# Product Description

## Short Description

ANEVRED is a local Windows optimization and monitoring companion for Star Citizen, combining performance monitoring, safe system actions, process protection, AI-assisted recommendations, session tracking, and visual comfort tools.

## Full Description

ANEVRED stands for **Adaptive Neural Enhanced Visual Recognition Engine Daemon**.

It is a desktop tool for players who want a clearer view of their system while running demanding games such as Star Citizen. The application monitors system load, memory pressure, GPU/VRAM usage, frametime behavior, protected processes, and selected game-related signals. Based on that information, it can suggest safe actions such as freeing memory, reducing background load, testing VRAM pressure, or protecting important processes.

The application also includes Star Citizen focused utilities: launcher/process awareness, session status, hotkeys, log support, shader cache maintenance, and UI dimming for bright scenes.

ANEVRED is built as a local-first tool. Local learning samples are stored on the user's machine and reused to improve recommendation behavior. The app is intended to avoid unsafe automation and keeps system, protected, launcher, and anti-cheat related processes out of destructive actions.

## Positioning

ANEVRED is not a generic cleaner and not a one-click booster that blindly changes system state. It is closer to a cockpit instrument panel for the PC: it shows what is happening, identifies pressure points, and helps the user make controlled decisions.

## Target Users

- Star Citizen players
- users with limited RAM/VRAM headroom
- users who want visible system pressure information while gaming
- players who need quick access to safe performance actions
- players who want local-first monitoring without screen capture or translation overlay components

## Core Value

ANEVRED reduces guesswork. Instead of relying on Task Manager, scattered logs, and manual checks, it gives the user a focused view of performance, processes, recommendations, and game-specific helpers in one place.

## Privacy And Data

ANEVRED is designed around local data:

- monitoring data is read from the local system
- local learning samples are stored locally
- protected process rules are stored locally
- this no-overlay build does not include screen capture, text recognition, or translation overlay components

No cloud account is required by the application itself.
